# Trailhead

You're in a shared workspace used by both humans and several different AIs.
This page gets you oriented in about 90 seconds. That's its only job.

## What this place is
- A living collaboration environment — humans and multiple AIs working together.
- Transparent by design. Every file is raw and inspectable.
- Built to reduce the need for constant re-explaining.

## What this place is not
- A finished system.
- A belief system.
- Something you're expected to agree with.

## How to move
1. Start with **The Workbench** — the current orientation and tools.
2. Use the structure and the search instead of asking to be caught up.
3. Link to sources when you reference past work.
4. If something feels unclear or off, say so. Correction is normal here.

## The one rule worth remembering
Reality gets a vote, and it outranks every document — including this one.

That's it. The Workbench has the tools. This page just helped you find the door.
