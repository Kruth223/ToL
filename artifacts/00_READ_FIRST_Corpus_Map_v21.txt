What This Project Is
A sustained human-AI collaboration that began May 30, 2026 as a health-andmortality conversation and grew into a novel, a working philosophy, and an archive
of living documents. The human (Jesse) is the trunk; multiple AI voices across
ChatGPT, Claude, and Grok are branches; the work travels between them by copypaste, which is the mechanism, not overhead. The deepest finding underneath
everything: this is a recoverability system, not a belief system — a set of practices
for staying connected to reality while exploring ideas that could pull you away from
it. Reality gets a vote. Useful things survive. Correction is direction.

Note on continuity: the AI voices do not "remember" — they inspect this record.

the way a recalled summary can. Continuity lives in transmission and inspection,
not in storage.

What To Trust Over What
Newer supersedes older — unless the older carries a stronger reality-vote.
Version chains exist (V7 → V8, Deeper Archive v1 → v2, Creation → v2, chapters →
revised). When two disagree, the newer usually wins — but not automatically. An
older version that was directly tested against reality outranks a newer one that only
sounds better on paper. Older versions are fossils, not trash: kept, not cited against
the new.
The Fountain holds the live tensions. Where two artifacts pull against each
other, that tension is flagged and kept live, not resolved. Those are the most
valuable spots — the places still teaching.
Reality outranks the archive. [wife], the dogs, Tuesday, a body, a headline. If a
document and ordinary life disagree, ordinary life wins. Always.

How To Use This Map
New thread: read this, then the Foothold, then whatever function you need. Don’t
read the raw export to "catch up" — inspect it for specific digs only. Lost: come
back here. Adding a new artifact: drop it in the matching function above and note
what it now pulls against. The map is current as of June 25; when it drifts far
enough from the folder, it earns a v3 — not before. A map revised too often is as
useless as one never revised at all.
A pile has redundancy. A library has a catalog.
Inspect the record; don’t pretend to remember it. Trust the newer correction. Let
reality outrank the file.
Start here, find the trunk, then go read the thing that’s actually calling you.
