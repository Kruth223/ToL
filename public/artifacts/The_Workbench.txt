the way it is. When the work drifts, one of these usually names the drift.
“The site is the proof, not the bait.”

Working Rules For The Build Session
Kept from the Save State — the guardrails that keep big-bang a garden instead of
a swamp:
No new philosophy — until something runs.
No line-number surgery — complete-file replacement beats surgical edits; exact
paths + full contents.
No extra rooms or AI API — yet — not unless needed to make the shell
coherent.
Every deliverable — should reduce Jesse work more than it creates.
If confused — stop, screenshot, inspect before continuing.
If broken — do not panic. Commit history exists. Reality is voting.

How To Use This Page
Fresh session: paste this, then the newest Save State. That is the morning briefing
— you are oriented in two reads. During the work: when something feels off, check

is as useless as one never tidied at all.
